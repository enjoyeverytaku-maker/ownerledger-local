# CSV Import Format

銀行入金 CSV は、最初は汎用形式として列の対応をユーザーが指定する設計です。

## 対応予定列

- 入金日
- 金額
- 振込名義
- 摘要
- 口座名

## 重複検知

次の組み合わせが同じ場合は重複候補として扱います。

- 入金日
- 金額
- 振込名義
- 摘要

重複候補は自動取込せず、確認画面に表示します。

## 例

`docs/examples/bank-payments.csv` を参照してください。

## 現在の実装

現在の取込画面では、次の列名を持つ CSV を取り込めます。

- `入金日`
- `金額`
- `振込名義`
- `摘要`
- `口座名`

取込前に確認ダイアログを表示し、同一日付・同一金額・同一名義・同一摘要の重複候補は保存しません。列マッピングを画面上で自由に指定する完全版ウィザードは次フェーズで実装します。
