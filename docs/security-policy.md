# Security Policy

## GitHub に上げないもの

- 本番 SQLite DB
- 添付ファイル
- バックアップ ZIP
- `.env`
- 個人情報入り CSV
- 銀行 CSV
- 契約書 PDF
- 領収書画像
- 保証会社資料
- 税理士提出用の本番資料
- 実在の入居者名や連絡先を含むファイル

## 個人情報

入居者情報は必要最小限を保存します。将来的な暗号化に備え、個人情報カラムを明確に分けています。

## 添付ファイル

添付ファイルは元ファイルを直接参照せず、アプリ専用フォルダへコピーして管理します。DB には表示名、保存場所、MIMEタイプ、サイズ、SHA-256 を保存します。SHA-256 は改ざん検知の補助情報であり、法的な真正性を保証するものではありません。

## 税務

アプリは税務判断を断定しません。税務上の取扱いは税理士に確認してください。
